## Visão Geral
Dashboard institucional com score de mercado, alertas Telegram, backtest e análise on-chain.

## Funcionalidades
- Score institucional em tempo real (0–100)
- Gráficos de candlestick com lightweight-charts
- Alertas automáticos via Telegram
- Backtest com engine V13
- Indicadores macro e on-chain

## Tecnologias
- HTML5 / CSS3 / JavaScript (ES6+)
- Chart.js, Lightweight Charts
- Binance API, CoinGecko, FRED, etc.
